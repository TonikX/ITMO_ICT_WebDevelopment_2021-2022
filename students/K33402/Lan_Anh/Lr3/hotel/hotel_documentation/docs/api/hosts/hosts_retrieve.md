# Retrieve Single host API

This API is meant to perform retrieve single operation with hosts.

**URL** : `/api/hosts/:id`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

### Success Responses

**Code** : `200 OK`

**Content** : `{[]}`
