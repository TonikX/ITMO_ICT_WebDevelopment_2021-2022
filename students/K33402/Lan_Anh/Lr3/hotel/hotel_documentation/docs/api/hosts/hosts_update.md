# update Single host API

This API is meant to perform update single operation with hosts.

**URL** : `/api/hosts/:id/update`

**Method** : `PUT`

**Auth required** : YES

**Permissions required** : None

### Success Responses

**Code** : `200 OK`

**Content** : `{[]}`
