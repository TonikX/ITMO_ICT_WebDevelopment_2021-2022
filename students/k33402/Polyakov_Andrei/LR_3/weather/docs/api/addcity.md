# Добавить город в список городов пользователя

Add city to user cities list.

**URL** : `/cities/preferences/`

**Method** : `POST`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `201`

**Content** : `{}`

```json
{
  "id": 0,
  "city": 0
}
```
