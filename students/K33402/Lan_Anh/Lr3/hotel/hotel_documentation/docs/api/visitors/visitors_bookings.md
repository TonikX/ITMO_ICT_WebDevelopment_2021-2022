# Retrieve visitor bookings API

This API is meant to perform retrieve operation with visitors bookings.

**URL** : `/api/visitors/:id/bookings`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

### Success Responses

**Code** : `200 OK`

**Content** : `{[]}`
