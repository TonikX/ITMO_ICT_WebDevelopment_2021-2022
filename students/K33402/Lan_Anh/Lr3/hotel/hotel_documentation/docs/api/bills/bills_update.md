# update Single bill API

This API is meant to perform update single operation with bills.

**URL** : `/api/bills/:id/update`

**Method** : `PUT`

**Auth required** : YES

**Permissions required** : None

### Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
{
    "id": 1,
    "service_name": "Stay",
    "description": "stay cost",
    "total": 160,
    "booking": 1
}
```
