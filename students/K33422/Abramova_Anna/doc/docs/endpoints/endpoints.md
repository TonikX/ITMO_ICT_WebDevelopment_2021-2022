# Endpoints

## ``` /admins/ ```

Доступные методы:

* get - получить список администраторовов
* post - создать администраторов

## ``` /admins/:id/ ```
Доступные методы:

* get - получить администраторов
* update - изменить администраторов
* patch - изменить администраторов
* delete - удалить администраторов
