# Показать список всех умений

Выводит информацию обо всех умениях.

**URL** : `skills/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

**Success info** : HTTP 200 OK

**Content-Type** : application/json

**JSON** :
```json
{
    "Skills": [
        {
            "title": "Singing"
        },
        {
            "title": "Dancing"
        }
    ]
}
```
