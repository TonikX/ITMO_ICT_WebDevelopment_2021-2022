# update Single hotel API

This API is meant to perform update single operation with hotels.

**URL** : `/api/hotels/:id/update`

**Method** : `PUT`

**Auth required** : YES

**Permissions required** : None

### Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
{
    "id": 1,
    "name": "Marina Bay Sand",
    "address": "singapore",
    "description": "marina bay sand is a good hotel",
    "owner": 4
}
```
