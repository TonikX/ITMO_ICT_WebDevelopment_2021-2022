# Create chicken

Creating a new chicken

**URL** : `/chickens/create/`

**Methods** : `POST`


## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`


```json
{
        "id": 1,
        "breed": "big chicken",
        "cage": "12",
        "weight": 5,
        "age": 1,
        "egg_amount": 5
    }
```
