# List of all breeds

Shows information about all breeds

**URL** : `/breeds/list/`

**Method** : `GET`


## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
[
    {
        "id": 1,
        "breed": "small chicken",
        "avg_eggs": 3,
        "avg_weight": 2,
        "diet": "table 5"
    },
    {
        "id": 2,
        "breed": "big chicken",
        "avg_eggs": 6,
        "avg_weight": 4,
        "diet": "table 1"
    }
]
```
