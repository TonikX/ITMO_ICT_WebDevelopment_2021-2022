# Список городов пользователя

Get users cities list.

**URL** : `/cities/preferences/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

**Code** : `200`

**Content** : `[{}]`

```json
[
  {
```
