# Показать всех воинов

Get all information about guest.

**URL** : `/api/guests/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
{
   "Status": [
       {
        "id": 1,
        "name": "e"
    },
    {
        "id": 2,
        "name": "f"
    }
   ]
}
```
