login(state, authData)

Делает POST запрос на сервер запрашивая токен авторизации. Поле authData имеет следующий вид { username: string, password: string } В ответе ожидает токен, который сохраняется в localstorig А также записывает токен в поле token состояения приложения После чего редиректит в личный кабинет. В случае неудачной попытки запроса, ожидает responseError, которой сохраняется в loginError, для дальнейшего вывода в виде особщения пользователю
